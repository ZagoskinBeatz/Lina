# Системные команды Linux — Краткая справка

## Работа с файлами
- ls — список файлов
- ls -la — подробный список (включая скрытые)
- cat файл — вывести содержимое файла
- head -n 20 файл — первые 20 строк
- tail -n 20 файл — последние 20 строк
- find /путь -name "*.txt" — поиск файлов
- grep "текст" файл — поиск текста в файле
- cp источник назначение — копировать файл
- mv источник назначение — переместить/переименовать
- mkdir имя — создать директорию

## Мониторинг системы
- htop — интерактивный менеджер процессов
- free -h — информация о RAM
- df -h — информация о дисках
- top — загрузка CPU и процессы
- ps aux — список процессов
- uptime — время работы системы

## Сеть
- ping адрес — проверка связи
- curl URL — HTTP-запрос
- wget URL — скачивание файла
- ip addr — сетевые интерфейсы
- ss -tlnp — открытые порты

## Пакеты (Arch/Pacman)
- sudo pacman -S пакет — установить пакет
- sudo pacman -Syu — обновить систему
- pacman -Ss запрос — поиск пакетов
- pacman -Qi пакет — информация о пакете

## Git
- git status — статус репозитория
- git add . — добавить все изменения
- git commit -m "сообщение" — создать коммит
- git push — отправить изменения
- git pull — получить изменения
- git log --oneline — история коммитов
